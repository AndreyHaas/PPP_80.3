# Циклы `while` и `do-while`

## Цикл `while`

Цикл `while` выполняет тело цикла **пока условие истинно**.  
Проверка условия выполняется **до** входа в цикл.

### Пример (Java):

```java
int i = 0;

while (i < 5){
    System.out.println(i);
    i++;
}
```
Особенности:

Тело может не выполниться ни разу, если условие сразу ложно.

Используется, когда количество итераций заранее неизвестно.

Цикл do-while

Цикл do-while выполняет тело минимум один раз, потому что условие проверяется после выполнения тела.

### Пример (Java):
```java
int i = 0;

do {
    System.out.println(i);
    i++;
} while (i < 5);
```
Особенности:

Гарантированно выполняется одна итерация, даже при ложном условии.

Удобен, когда нужно сначала выполнить действие, а потом проверять условие (например, вывод меню).

Сравнение

while: сначала проверка → потом выполнение (минимум 0 раз).

do-while: сначала выполнение → потом проверка (минимум 1 раз).
